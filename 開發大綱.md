# What to eat !!
> **專案名稱:** WTE
>
> **用途：** 一到吃飯時，煩惱著吃什麼，使用 WTE 系統，來定位自身位置，根據條件從 google map 中篩選想吃的餐廳，並從中隨機選一家挑選。

## 篩選條件
1. 離自身定位距離，單位公尺 (1個 input，預設 1000公尺，必填) 
2. 價格區間 (2個 input，非必填)
3. 評等區間 1~5 星，可支援小數點後一位 (2個 input，非必填)
4. 評論數，至少大於 X 條，用來篩選是否很多人吃過 (1個 input，非必填)

## 業務邏輯
1. 需要註冊會員，登入後才可以使用
2. 使用篩選條件，隨機選出一家餐廳，使用彈窗顯示，並且有重新挑選按鈕。
3. 餐廳彈窗有以下資訊 (有的話才提供)
- 店名
- 價格
- 評等
- 距離自身定位距離 (公尺)
- 地址
- 店家電話
- google map 餐廳圖片
4. 選中餐廳歷史紀錄，按照日期排序

## 主要畫面
1. 登入頁
2. 註冊頁
3. 主頁面(一頁式網站)，從上至下的區塊為
- 堪入 google map
- 篩選條件
- 選中餐廳歷史紀錄
4. 選中餐廳彈窗

## 程式架構
1. 後端使用 php 8.2.28，laravel 12
2. 前端使用 vue 最新的穩定版本
3. 前後分離，使用 restful api
4. db 使用 mysql:8.0.34
5. 使用 docekr 部署 (需有docker-compose)
